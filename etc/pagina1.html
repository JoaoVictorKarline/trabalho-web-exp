O que é JavaScript?

JavaScript é uma linguagem de programação versátil e poderosa que traz interatividade aos navegadores. Criado em 1995, tornou-se a linguagem padrão para desenvolvimento web no lado do cliente (frontend), mas também é utilizada no servidor (backend) através de plataformas como Node.js.

Diferentemente do HTML (que define a estrutura) e do CSS (que cuida da apresentação), JavaScript é responsável pelo comportamento e interatividade das páginas web.

JavaScript: A linguagem da interatividade web

Características principais do JavaScript:

  • Dinâmico: Executa em tempo real dentro do navegador
  • Orientado a Objetos: Suporta programação orientada a objetos
  • Funcional: Permite programação funcional e de ordem superior
  • Assíncrono: Suporta operações não-bloqueantes
  • Leve: Não requer compilação, funciona nativamente nos navegadores

Por que JavaScript é essencial?

JavaScript transformou a web de um meio estático para um ambiente dinâmico e interativo. Hoje em dia, é impossível imaginar um site moderno sem JavaScript.

O que você pode fazer com JavaScript?

1. Manipulação do DOM (Document Object Model)

JavaScript permite acessar e modificar os elementos da página em tempo real:

// Selecionar um elemento const titulo = document.getElementById('titulo'); // Modificar seu conteúdo titulo.textContent = 'Olá, JavaScript!'; // Alterar estilos titulo.style.color = '#667eea'; titulo.style.fontSize = '2rem';

2. Tratamento de Eventos

Responda às ações do usuário como cliques, digitação, movimento do mouse, etc:

// Adicionar um evento de clique const botao = document.querySelector('button'); botao.addEventListener('click', function() { alert('Botão clicado!'); console.log('Usuário interagiu com a página'); });

3. Validação de Formulários

Valide dados do usuário antes de enviar para o servidor:

// Validar email function validarEmail(email) { const padrao = /^[^\s@]+@[^\s@]+\.[^\s@]+$/; return padrao.test(email); } if (validarEmail('usuario@exemplo.com')) { console.log('Email válido!'); }

4. Requisições Assíncronas (AJAX/Fetch)

Comunique-se com o servidor sem recarregar a página:

// Buscar dados do servidor fetch('https://api.exemplo.com/dados') .then(resposta => resposta.json()) .then(dados => console.log(dados)) .catch(erro => console.error('Erro:', erro));

5. Animações e Efeitos

Crie animações suaves e efeitos visuais:

// Animar um elemento const elemento = document.querySelector('.box'); let posicao = 0; function animar() { posicao += 2; elemento.style.transform = `translateX(${posicao}px)`; if (posicao < 300) { requestAnimationFrame(animar); } } animar();
